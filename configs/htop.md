# htop options

## Display options

- Tree View (F5) - *will sort in groups, rather than across all processes*
- [x] Highlight program "basename"
- [x] Update process names on every refresh
- [x] Also show CPU temperature
- [x] Highlight new and old processes
- [3] Highlight time
